# Event Storming UI

This project is a modern TypeScript-based frontend application for event storming sessions.

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

4. **Preview the production build**
   ```bash
   npm run serve
   ```

## Features
- React with TypeScript
- Tailwind CSS for styling
- Vite for fast development and hot-reloading

## Component Structure
- `DraggableElement`: A basic component to demonstrate draggable functionality.
